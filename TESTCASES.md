1. Главная страница: Проверка фильтра «Диапазон цен»

ID: TC_HP_01
Название: Проверка фильтра «Диапазон цен» на главной странице
Предусловия: Пользователь открыл главную страницу сайта. Товары имеют ценовые значения.
Шаги:

Найти фильтр «Диапазон цен».
Установить минимальное и максимальное значение диапазона (например, 1000–5000).
Применить фильтр.
Ожидаемый результат:
На странице отображаются только товары с ценой в указанном диапазоне.
В интерфейсе отображаются выбранные значения фильтра.
Никакие товары за пределами диапазона не отображаются.

2. Главная страница: Проверка сортировки «По цене»

ID: TC_HP_02
Название: Проверка сортировки «По цене» на главной странице
Предусловия: Пользователь на главной странице, товары загружены.
Шаги:

Найти выпадающий список сортировки.
Выбрать сортировку «По цене: по возрастанию».
Проверить порядок отображения товаров.
Выбрать сортировку «По цене: по убыванию».
Проверить порядок отображения товаров.
Ожидаемый результат:
При сортировке по возрастанию цены товаров увеличиваются сверху вниз.
При сортировке по убыванию цены товаров уменьшаются сверху вниз.
Отображаемые товары соответствуют выбранному варианту сортировки.
3. Главная страница: Проверка фильтра «Категория»

ID: TC_HP_03
Название: Проверка работы фильтра «Категория»
Предусловия: Пользователь на главной странице, товары имеют категории.
Шаги:

Найти фильтр «Категория».
Выбрать одну категорию (например, «Электроника»).
Применить фильтр.
При желании выбрать несколько категорий одновременно.
Ожидаемый результат:
Отображаются только товары выбранной категории.
Если выбраны несколько категорий — отображаются товары всех выбранных категорий.
Товары, не относящиеся к выбранной категории(ям), не отображаются.
4. Главная страница: Проверка тогла «Только срочные»

ID: TC_HP_04
Название: Проверка работы тогла «Только срочные»
Предусловия: Пользователь на главной странице, есть срочные и обычные товары.
Шаги:

Найти переключатель (toggle) «Только срочные».
Активировать его.
Проверить отображение товаров.
Деактивировать переключатель.
Ожидаемый результат:
При включенном тогле отображаются только срочные товары.
При выключенном тогле отображаются все товары.
5. Страница Статистики: Контейнер управления таймером обновления

ID: TC_STAT_01
Название: Проверка кнопок управления таймером на странице статистики
Предусловия: Пользователь открыл страницу статистики, таймер активен или готов к запуску.
Шаги:

Найти контейнер управления таймером (кнопки «Обновить», «Стоп», «Старт»).
Нажать кнопку «Обновить».
Проверить, что данные статистики обновились.
Нажать кнопку «Стоп».
Проверить, что автоматическое обновление остановлено.
Нажать кнопку «Старт».
Проверить, что автоматическое обновление снова включено.
Ожидаемый результат:
Кнопка «Обновить» инициирует немедленное обновление статистики.
Кнопка «Стоп» останавливает таймер обновления.
Кнопка «Старт» запускает таймер снова.
Интерфейс корректно отображает текущий статус таймера (стоп/старт).
